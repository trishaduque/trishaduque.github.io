About This Gallery

This site explores the Art Institute of Chicago’s collection in two ways: a curated “What to See in an Hour” tour based on the museum’s highlight guide, and a live view powered by the Art Institute’s public API.

What to See in an Hour

In highlights mode, the page focuses on a small set of iconic works featured in the Art Institute’s “What to See in an Hour” guide . These include pieces by artists such as Georges Seurat, Vincent van Gogh, Georgia O’Keeffe... It’s essentially a digital version of a one-hour museum walk.

Live Collection View

In live mode, the site pulls artworks directly from the Art Institute’s API (https://api.artic.edu/api/v1/artworks). These API data dumps are updated nightly, so the selection you see can change over time as records are added or updated.

How the Data Is Shown

  • Card View: Visual tiles for browsing artworks one by one, with image and key details.
  • Table View: A sortable data table for scanning titles, artists, dates, and departments.
  • Category View: Groups works by museum department, like walking through different galleries.
  • Stats View: Summarizes the current selection with counts by artist, department, and more.
